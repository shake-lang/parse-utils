<TabItem value="JVM/Gradle" label="JVM/Gradle" default>

To install the JVM/Gradle package just add the dependency

<Tabs groupId="build-scripts">

<TabItem value="groovy" label="build.gradle" default>

```groovy
dependencies {
  implementation 'io.github.shakelang.parseutils:lib-jvm:$version'
}
```

</TabItem>

<TabItem value="kotlin" label="build.gradle.kts">

```kotlin
dependencies {
  implementation("io.github.shakelang.parseutils:lib-jvm:$version")
}
```

</TabItem>
</Tabs>

</TabItem>
<TabItem value="JVM/Maven" label="JVM/Maven">

Just add the maven dependency to your project.

```xml
<dependencies>
   <dependency>
      <groupId>io.github.shakelang.parseutils</groupId>
      <artifactId>lib-jvm</artifactId>
      <version>{version}</version>
   </dependency>
</dependencies>
```

</TabItem>

<TabItem value="JS">

Add the following dependency to your dependencies block:

<Tabs groupId="build-scripts">

<TabItem value="groovy" label="build.gradle" default>

```groovy
dependencies {
  implementation 'io.github.shakelang.parseutils:lib-js:$version'
}
```

</TabItem>

<TabItem value="kotlin" label="build.gradle.kts">

```kotlin
dependencies {
  implementation("io.github.shakelang.parseutils:lib-js:$version")
}
```

</TabItem>
</Tabs>

Alternatively, add the dependency to the Javascript specific target.

<Tabs defaultValue="groovy" groupId="build-scripts">

<TabItem value="groovy" label="build.gradle" default>

```groovy
kotlin {
   targets {
      js {
         browser()
         nodejs()
      }
   }
   sourceSets {
      val jsTest by getting {
         dependencies {
           implementation 'io.github.shakelang.parseutils:lib-js:$version'
         }
      }
   }
}
```

</TabItem>

<TabItem value="kotlin" label="build.gradle.kts">

```kotlin
kotlin {
   targets {
      js {
         browser()
         nodejs()
      }
   }
   sourceSets {
      val jsTest by getting {
         dependencies {
           implementation("io.github.shakelang.parseutils:lib-js:$version")
         }
      }
   }
}
```

</TabItem>
</Tabs>
</TabItem>
<TabItem value="Multiplatform">

      val commonMain by getting {
