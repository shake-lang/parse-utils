Installation

  • JVM/Gradle
  • JVM/Maven
  • JS
  • Multiplatform

To install the JVM/Gradle package just add the dependency

  • build.gradle
  • build.gradle.kts
dependencies {  implementation 'com.github.shakelang.parseutils:lib-jvm:$version'}
dependencies {  implementation("com.github.shakelang.parseutils:lib-jvm:$version")}

Just add the maven dependency to your project.

<dependencies>   <dependency>      <groupId>com.github.shakelang.parseutils</groupId>      <artifactId>lib-jvm</artifactId>      <version>{version}</version>   </dependency></dependencies>

Add the following dependency to your dependencies block:

  • build.gradle
  • build.gradle.kts
dependencies {  implementation 'com.github.shakelang.parseutils:lib-js:$version'}
dependencies {  implementation("com.github.shakelang.parseutils:lib-js:$version")}

Alternatively, add the dependency to the Javascript specific target.

  • build.gradle
  • build.gradle.kts
kotlin {   targets {      js {         browser()         nodejs()      }   }   sourceSets {      val jsTest by getting {         dependencies {           implementation 'com.github.shakelang.parseutils:lib-js:$version'         }      }   }}
kotlin {   targets {      js {         browser()         nodejs()      }   }   sourceSets {      val jsTest by getting {         dependencies {           implementation("com.github.shakelang.parseutils:lib-js:$version")         }      }   }}

For multiplattform you can just add the dependency to your commonMain

  • build.gradle
  • build.gradle.kts
kotlin {   ...   sourceSets {      val commonMain by getting {         dependencies {           implementation 'com.github.shakelang.parseutils:lib:$version'         }      }   }}
kotlin {   ...   sourceSets {      val commonMain by getting {         dependencies {           implementation("com.github.shakelang.parseutils:lib:$version")         }      }   }}
